üì¶ Data Types & Casting

Primitive Types

str_var = "hello"       # String
int_var = 42         # Integer
float_var = 3.14     # Float
bool_var = True      # Boolean
none_var = None      # NoneType
bytes_var = b"data"  # Bytes

Type Conversion

str(42)        # "42"
int("10")      # 10
float("3.5")   # 3.5
bool("")       # False
list("abc")    # ['a', 'b', 'c']
tuple([1,2,3]) # (1, 2, 3)

Type Checking

type("hello") is str  # True
isinstance(42, int) # True

üîÅ Control Flow

Conditionals

if age > 18:
    print("Adult")
elif age == 18:
    print("Exactly 18")
else:
    print("Minor")

# Ternary operator
status = "Adult" if age >= 18 else "Minor"

Loops

# For loop
for i in range(5):       # 0 to 4
    print(i)

for item in [1, 2, 3]:   # Iterate list
    print(item)

# While loop
while condition:
    # Loop body
    if should_exit:
        break     # Exit loop
    if skip_rest:
        continue  # Next iteration

Loop Techniques

# With index
for i, value in enumerate(['a', 'b', 'c']):
    print(f"Index {i}: {value}")

# Multiple sequences
for a, b in zip([1, 2], ['x', 'y']):
    print(a, b)

# Dictionary iteration
for key, value in my_dict.items():
    print(key, value)

üßÆ Operators

Category Operators Description
Arithmetic + - * / % ** // Addition, subtraction, multiplication, division, modulo, exponent, floor division
Comparison == != > < >= <= Equal, not equal, greater than, less than, etc.
Logical and or not Boolean operations
Assignment = += -= *= /= Assignment operators
Identity is is not Object identity comparison
Membership in not in Check if value exists in sequence
Bitwise & | ^ ~ << >> Bitwise AND, OR, XOR, NOT, shifts
Watch Out: == checks equality, is checks if objects are identical (same memory location).
